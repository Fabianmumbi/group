In this example, the #menu-btn button is selected using document.getElementById, and an event listener is added to it using addEventListener. When the button is clicked, the paragraph element with id #my-paragraph is selected and its style.display property is set to either "block" or "none" depending on its current value. When the property is set to "none", the paragraph element will be hidden, and when it is set to "block", it will be shown again.
